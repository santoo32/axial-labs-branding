→ Spec

Font JetBrains Mono
Size 12px
Weight 500 (medium)
Tracking +6% (0.06em)
Case UPPERCASE
Default color graphite-400 #878C97
Prefix convention → (Mono arrow)
Margin below 12–16px before heading

→ HTML

<!-- Base eyebrow -->
<span class="eyebrow">→ Services</span>

<!-- With CSS-generated → prefix -->
<span class="eyebrow eyebrow--arrow">Services</span>

<!-- Voltage accent -->
<span class="eyebrow eyebrow--voltage">→ New</span>

<!-- Section counter: "→ 01 / 03  ·  Label" -->
<span class="eyebrow">→ 01 / 03 &nbsp;·&nbsp; Brand systems</span>

<!-- Paired with heading -->
<span class="eyebrow">→ Pricing</span>
<h1>Three tiers. Fixed scope.</h1>
