→ 02 / 02   Usage

→ How to use these files

Each component file is a self-contained HTML preview. Open it in a browser to see the component in isolation. Find the HTML and CSS code sections at the bottom of each file — copy and paste into your project. Token variables must be present in :root (use /tokens/build/tokens.css as your source of truth).

→ Token dependency

All components reference CSS custom properties. Include tokens.css before any component stylesheet. Components work without Tailwind — they use vanilla CSS only.

<link rel="stylesheet" href="/tokens/build/tokens.css" />
<!-- then your component styles -->

→ Axial element requirement

Every composed page must include at least one axial element: the fixed vertical axis line, tick marks, a crosshair, or a Mono index label. Individual components contribute to this — the nav's axis line, eyebrow labels, and table tick marks all count. If your composition has none, add one.
